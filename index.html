Welcome to ShoeBox!

ShoeBox is a collection of open-source libraries by Matthew Shoemaker. Each library is designed with a focus on quality, performance, and simplicity. A strong emphasis is placed on ease of use, minimal learning curve, and clean APIs with which consumers interact.
