Welcome to ShoeBox

ShoeBox is a collection of open-source libraries by Matthew Shoemaker. Each library is designed with a focus on quality, performance, and usability.

Libraries

Kernl

A library for abstracting cache/data logic in the data layer with a declarative approach.

Kotlin Caching

Stratify

A KSP plugin to facilitate the generation of code through strategy patterns.

Kotlin KSP

AutoBuilder

A Kotlin library that simplifies the creation of builder patterns with auto-generated code.

Kotlin Builder Pattern KSP

MultiVers

A Gradle plugin that automates testing and validation of dependencies across multiple versions.

Gradle Plugin Version Ranging

Pasty

An IDE plugin to automate interactions with Android emulators using ADB commands.

Intellij Plugin Android ADB

Guides

Comprehensive guides and tutorials are available for each library in the ShoeBox collection.

About ShoeBox

ShoeBox is a personal open-source initiative by Matthew Shoemaker. The goal is to provide high-quality, well-documented libraries to the developer community.
